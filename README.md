# 🏏 IPL Data Analysis (2008–2024) | Exploratory Data Analysis Using Python

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on Indian Premier League (IPL) data from **2008 to 2024** using Python. The objective is to analyze player performances, team performances, match outcomes, and venue statistics by transforming raw ball-by-ball data into meaningful insights through data cleaning, feature engineering, and visualization.

---

## 🎯 Problem Statement

The Indian Premier League (IPL) generates a massive amount of ball-by-ball data every season. Understanding player performances, team strategies, match outcomes, and venue impact can be challenging due to the volume and complexity of the data.

This project applies Exploratory Data Analysis (EDA) techniques to uncover meaningful trends and insights that support data-driven decision-making.

---

## 📂 Dataset

The project uses two datasets:

* **matches.csv** – Match-level information
* **deliveries.csv** – Ball-by-ball delivery data

**Dataset includes:**

* Match Details
* Team Information
* Toss Details
* Player Statistics
* Batting Performance
* Bowling Performance
* Match Results
* Venue Information

## 📊 Exploratory Data Analysis

The following analyses were performed:

1. Orange Cap Winners (Highest Run Scorers)
2. Players with Most Sixes in Each Season
3. Purple Cap Winners (Highest Wicket Takers)
4. Bowlers with Most Dot Balls
5. Highest Target in Each IPL Season
6. Lowest Target in Each IPL Season
7. IPL Titles Won by Each Team
8. Overall Win Percentage: Batting First vs Chasing
9. Fastest Fifty in Each IPL Season
10. Top 10 Venues Hosting the Most IPL Matches
11. Toss Winner vs Match Winner Analysis
12. Players with the Most Player of the Match Awards

---

## 📈 Key Insights

* Batting performances have become more aggressive in recent IPL seasons.
* Chasing teams have won a significant number of matches.
* Winning the toss provides some advantage but does not guarantee victory.
* Certain players and teams have consistently dominated the IPL.
* Venue conditions play an important role in match outcomes and scoring patterns.
